About this project

This is a webApp built with Angular 7 for the front end, not fully responsive, and most foccused on learning the functionality between components, services, HTTP requests, libraries and backend using NodeJs express. The purpose of this webApp is to create projects/products with their information and display them in a organised way to be able to add, edit or remove them at anytime with just a few clicks, all the functionality is instantly with no refreshing of the app, similar to a control panel functionality.
